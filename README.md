## ✅ Task 3 - Student Record Management System - COMPLETED
A C program to manage student records using CRUD operations with file handling.

### ✨ Features
- Add Student - Store roll no, name, marks in binary file
- Display All Students - Read from file and show all records
- Search Student - Find student by roll number
- Update Student - Modify existing student details
- Delete Student - Remove student record from file
- Data saved permanently in 'students.dat'

### 💻 How to Compile & Run
gcc "Task3 Student Management System" -o student_system
./student_system
